# 파일 관련 명령어

2023-03-21



## Directory : 절대/상대 경로

### Absolute Path(절대경로)

- root 디렉터리(/)를 시작으로 하는 경로

  e.g. /usr/local/bin

### Relative Path(상대경로)

- 현재 디렉터리(.)를 시작으로 하는 경로

  - dot(.)는 생략 가능

  e.g) ../../tmp (./../../tmp와 같음)
